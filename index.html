Food Fit

Food Fit is a Java Android App that gives you access to 5000+* recipes, each with nutritional information. Users can search for or view random recipes. Recipes are fetched from an API. The app uses Firebase for Authentication: login, signup and password reset.

* Although the App has capacity to display 5000+ recipes, I am using a free plan of the API with a capped number of recipe-requests per month.

This is python Django blog that allows authenticated users to write and post blog articles. Authors are allowed to create, read, update or even delete their articles (CRUD). This web application allows any logged in user to leave a comment for each article. There is also a reset password feature for users who forget their passwords. Styling is done by bootstrap.

The admin has rights to edit or delete any article or comment.

The App uses SQLite Database for the development version, and PostgreSQL for the deployed version.
